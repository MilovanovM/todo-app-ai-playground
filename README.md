# TO-DO list

This repo contains implementations of a simple to-do list app in different languages and frameworks.

The apps were created via Cursor and use the same UI (based on Bootstrap).

**Backends**

- ASP.NET (Web API)
- Go
- Python (Django and FastAPI)
- PHP (Laravel)
- Ruby (Rails)
- NodeJS (Express)
- Java

**Web Frontends**

- React
- Angular
- Blazor
- Vue
- Pure JavaScript

**Mobile Frontends**

- React Native
- Flutter
- MAUI
- Swift
- Kotlin
